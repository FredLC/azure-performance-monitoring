# Enhancing Application Reliability & Performance on Azure

This project showcases the implementation of **monitoring, auto-scaling, and automated remediation** for a cloud-based application using Microsoft Azure.

The focus was on building a system that can **detect performance issues, scale dynamically, and self-heal** with minimal manual intervention.


## Overview

Designed an end-to-end observability and automation solution using:

- Azure Kubernetes Service (AKS)  
- Virtual Machine Scale Sets (VMSS)  
- Application Insights & Log Analytics  
- Azure Automation (Runbooks)


## Key Features

### Monitoring & Telemetry
Integrated Application Insights to collect and analyze application and infrastructure metrics.

### Auto-Scaling
Configured VMSS and AKS scaling based on performance thresholds.

### Automated Remediation
Built Azure Runbooks to automatically resolve performance issues.

### Alerting
Created alerts to trigger scaling actions and remediation workflows.


## Outcome

- Improved application **reliability and scalability**  
- Enabled **automated incident response**  
- Reduced manual intervention through **self-healing mechanisms**
